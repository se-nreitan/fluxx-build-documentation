=== Fluxx Build Documentation Data Scraper ===

This tool extracts model, theme, and view information from Fluxx and generates both console output and a Word document.

Features:
- Automated navigation to Admin Panel
- Extraction of all models, themes, and views
- Generation of formatted Word document with timestamp
- Console output for real-time progress

Output:
- Console display of extracted information
- Word document (fluxx_documentation_YYYYMMDD_HHMMSS.docx) containing formatted documentation

Requirements:
- Google Chrome browser
- Internet connection
- Fluxx admin credentials

Usage:
1. Run the executable
2. Enter your Fluxx URL
3. Log in when prompted
4. Wait for extraction to complete
5. Find the generated Word document in the same folder as the executable

The Word document will be named 'fluxx_documentation_YYYYMMDD_HHMMSS.docx' with the current timestamp.

Instructions:
1. Double-click Fluxx_Scraper.exe to run the program
2. Enter your Fluxx site URL when prompted
3. Enter your login credentials when prompted
4. Follow the on-screen instructions

Note: This application includes a portable version of Chrome and does not require any installation.
